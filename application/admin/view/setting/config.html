快递公司列表数据模式：
1：快递公司指定接口默认模式则后台快递公司选择哪一个接口则返回哪一个接口的渠道
2：快递公司指定接口比价模式则是全部筛选出来全部渠道并按照价格最低排序【建议最多同时开启3个接口】
3：快递公司全部接口展示模式则会出现多个快递公司【建议最多同时开启3个接口】
4：快递公司全部接口全部渠道展示模式则展示出来全部渠道这样一个快递公司会展示多个渠道，当前模式下面显示渠道详细名称【建议最多同时开启3个接口】
特别说明：一般选择1或者2那么3跟4按需选择
快递公司列表排序模式：
1：后台快递公司升序模式则是按照我们后台编辑快递公司排序值大小升序排序适合于大部分场景
2：支付价格升序模式则是让用户从低价格选择到高的价格
3：支付价格降序模式则是让用户选择从高价格到低价格
4：揽件率升序模式一般不会选择建议非要选择可以选择揽件率从高到低避免退货
5：随机模式一般不选择用处不大
