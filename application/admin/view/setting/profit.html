注意：1：这里开通分销，2：这页面配置分成比例，3：用户确认收货后即可给上级分成

基本设置
开启分销功能： 这里暂时没用，如果你不想分成，则在这里订单分成费率都填写0
分成方式：
1：城市代理+传统3级分销：后台导入城市后，一个城市可配置一个代理商，当前城市的用户下单对应的区县，城市，省份代理有分成可配置极差
2：等级代理分成模式：后台用户等级为代理级别，高等级的代理可添加下级代理，可编辑代理分成费率，在一条线上可实现多级分销，可配置对应等级额外奖励
3：城市代理+传统3级分销+等级代理分成：以上2种功能都兼容
4：城市代理外链：/pages/common/content/web?url={$CONFIG.site.host}/agent
5：等级代理外链：/pages/common/content/web?url={$CONFIG.site.host}/retail
拥有分销权限用户最低等级：
城市代理分成模式：
1：单代理分成，代理商在后台，城市，编辑设置代理商的会员ID
2：多代理分成，代理商在后台，城市，添加代理，添加多个会员ID
直推奖励
是否开启： 全局开启关闭功能此功能默认关闭
直推用户下单多少： 直推用户下单多少单后才有奖励
直推用户奖励金额： 直推用户下单多少单后才有奖励金额
分成原价计算模式
分成模式： 这里妥善选择，如果你选择分利润，那么你分成费率可以搞一些，如果你选择分订单总金额，分成费率可以低一些
分成价格计算模式： 分成价格计算模式按照费率跟按照固定价格是不一样的
一级订单分成比例： % 1级分成比例，不填写或者填写0不分成
二级订单分成比例： % 2级分成比例，不填写或者填写0不分成
三级订单分成比例： % 3级分成比例，不填写或者填写0不分成
一级固定分成金额： 元 一级固定分成金额
二级固定分成金额： 元 二级固定分成金额
三级固定分成金额： 元 三级固定分成金额
VIP购买分成设置
VIP购买一级订单分成比例： % VIP购买1级分成比例，不填写或者填写0不分成
VIP购买二级订单分成比例： % VIP购买2级分成比例，不填写或者填写0不分成
VIP购买三级订单分成比例： % VIP购买3级分成比例，不填写或者填写0不分成
区县代理配置
开启区县分成： 全站开启区县分成
区县代理默认费率： % 区县代理默认费率如果区县代理商那么没配置则调用这里的费率【优先级1：区县编辑里面设置的费率2：区县代理默认费率】
推荐人是VIP区县代理默认费率： % 如果当前订单的推荐人是VIP情况下，区县代理的分成费率【优先级1：区县编辑里面设置的VIP费率2：推荐人是VIP区县代理默认费率】
城市代理配置
开启城市分成： 全站开启城市分成
城市代理默认费率： % 城市代理默认费率如果城市代理商那么没配置则调用这里的费率【优先级1：城市编辑里面设置的费率2：城市代理默认费率】
推荐人是VIP城市代理默认费率： % 如果当前订单的推荐人是VIP情况下，城市代理的分成费率【优先级1：城市编辑里面设置的VIP费率2：推荐人是VIP城市代理默认费率】
代理极差配置
开启极差功能：
比如我设置市代分配 30% ，区代分配50%，在没有区代、只有市代的情况下，市代独享80%
比如市代下面，有区代出现的时候，那么系统按照正常的设置结算，市代分配30%，区代分配50%
